# 💖 One Year Anniversary

A custom-built interactive website created to celebrate our first year together.

This project was designed from the ground up as a virtual world that two players can explore together. Every location represents a memory, a milestone, or a special part of our relationship.

---

## 🌸 Features

### 🏡 Main World
The main hub of the experience where players can walk around together and visit different locations.

### 🏠 Main Cabin
A cozy cabin filled with memories, letters, photos, future plans, and special surprises.

### 🌸 Cherry Blossom Lake
A peaceful lakeside area inspired by quiet moments together and beautiful memories.

### 🚁 Helipad
A place dedicated to adventures, travel dreams, and future destinations.

### 🛍️ Shopping Center
A fun area filled with gifts, surprises, and future additions.

### 💕 Couple Features
- Two-player movement
- Pixel-art style animations
- Dynamic heart effect when standing together
- Interactive memory locations
- Custom artwork and maps
- Hidden surprises throughout the world

---

## 🎮 Controls

### Player One
- Arrow Keys = Move

### Player Two
- W A S D = Move

### Interaction
- E = Interact
- G = Toggle Debug Mode
- B = Box Selection Tool (Debug)

## ❤️ Why This Exists

This website was created as a one-year anniversary gift.

Instead of a traditional scrapbook, I wanted to build something interactive that could grow over time. Every location, memory, and future update represents a part of our story together.

The goal is simple:

To create a place where our memories can live forever.
